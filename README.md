1) Исследование данных о продаже компьютерных игр
   <u>Описание проекта</u> По историческим данным интернет-магазина, продающего компьютерные игры по всему миру, необходимо было выявить определяющие успешность игры закономерности, чтобы сделать ставку на потенциально прибыльный продукт и правильно спланировать рекламные кампании.
